## Installing Pug

**Pug**란 **view engin**이다. 설치해보자
```
npm install pug
```
expressjs [docs](https://expressjs.com/en/5x/api.html#app.set) 에서 api 부분에 **app.set(name,value)**을 보면 된다.   

## app.set(name,value)
이것을 통해서 view engin 설정을 바꿔줄것이다. view engine설정의 기본값은 undefined이다.
